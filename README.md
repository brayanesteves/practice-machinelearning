# Machine Learning Practice / Práctica de Machine Learning

This repository is dedicated to practicing various Machine Learning concepts and algorithms.
Este repositorio está dedicado a la práctica de diversos conceptos y algoritmos de Machine Learning.

---

## 🇺🇸 English Version

### 🛠 Technology Stack
The project primarily uses the following libraries:
- **Python**: Core programming language.
- **Pandas**: For data manipulation and analysis.
- **NumPy**: For numerical computing.
- **Scikit-learn**: For machine learning algorithms and preprocessing tools.

### Environment Setup (using `venv`)
It is recommended to use a virtual environment to manage dependencies and avoid conflicts.

1.  **Create the environment**:
    ```powershell
    python -m venv venv
    ```
2.  **Activate the environment**:
    - **Windows**:
      ```powershell
      .\venv\Scripts\activate
      ```
    - **Linux/macOS**:
      ```bash
      source venv/bin/activate
      ```

### Installing Packages per Module
This repository is organized into specific folders for different topics. To install the dependencies for a specific module, navigate to that directory and run:

```powershell
pip install -r requirements.txt
```

### 🧠 Required Knowledge
To effectively work through these exercises, the following knowledge is recommended:
- **Python Programming**: Familiarity with loops, functions, and basic data structures.
- **Mathematics for ML**:
    - **Linear Algebra**: Matrices and vectors.
    - **Statistics & Probability**: Distributions, mean, variance, and hypothesis testing.
    - **Calculus**: Basic understanding of derivatives (for gradient descent).
- **Data Manipulation**: Understanding how to clean and prepare data using Pandas.
